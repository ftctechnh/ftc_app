Scan the keypad for pressed keys. This returns immediately.

Parameters:
linkthe MSNP port number
pressedKeysa binary representation of all keys pressed
keycharacter representation of the first key pressed, X if no key was pressed
numberthe numeric equivalent, 0-9 for digits, -1 for # and -2 for *, -255 when nothing is pressed
Returns:
true if no error occured, false if it did
