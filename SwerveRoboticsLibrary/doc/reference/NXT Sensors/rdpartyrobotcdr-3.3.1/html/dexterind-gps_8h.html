Detailed Description

Dexter Industries GPS Sensor driver.

DGPS-driver.h provides an API for the Dexter Industries GPS Sensor.

Changelog:

  • 0.1: Initial release
  • 0.2: Added DGPSreadDistToDestination()
  • 0.3: Changed from array structs to typedefs
    Fixed typos and ommissions in commands
