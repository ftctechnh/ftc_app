bool MSTPgetTouch ( tSensors  link,
int &  x,
int &  y,
ubyte &  buttons,
ubyte  addr 
)

Fetch all the information from the TouchPanel, including coordinates of current touch and button state. by configuring the sensor as sensorLightInactive

Parameters:
linkthe TouchPanel port number
xthe X coordinate of the current pen position, 0 if nothing is detected
ythe Y coordinate of the current pen position, 0 if nothing is detected
buttonsthe currently touched buttons, use isButtonXXTouched() to check the individual status.
addrthe I2C address of the TouchPanel, is optional, defaults to 0x04
Returns:
true if no error has occured, false if it did
Examples:
mindsensors-touchpanel-test1.c.
