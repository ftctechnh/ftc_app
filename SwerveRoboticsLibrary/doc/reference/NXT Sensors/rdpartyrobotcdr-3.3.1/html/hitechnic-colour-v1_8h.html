Detailed Description

HiTechnic Color Sensor driver.

hitechnic-colour-v1.h provides an API for the HiTechnic Color Sensor driver.

Changelog:

  • 0.1: Initial release
  • 0.2: Added SMUX functions
  • 0.3: All functions that uses tIntArray are now pass by reference to reduce memory usage.
    Removed SMUX data array
  • 0.4: Use new calls in common.h that don't require SPORT/MPORT macros
    Removed calls to ubyteToInt()
  • 0.5: Replaced array structs with typedefs

Credits:

  • Big thanks to HiTechnic for providing me with the hardware necessary to write and test this.
