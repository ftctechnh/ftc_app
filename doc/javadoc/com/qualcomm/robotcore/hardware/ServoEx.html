Interface ServoEx

  • All Known Implementing Classes:
    ServoImplEx

    public interface ServoEx
    The ServoEx interface provides enhanced servo functionality which is available with some hardware devices. The ServoEx interface is typically used as a second interface on an object whose primary interface is Servo or CRServo. To access it, cast your Servo or CRServo object to ServoEx. However, it is perhaps prudent to first test whether the cast will succeed by testing using 'instanceof'.
